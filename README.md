# Лабораторные работы по Java

Готовый репозиторий с лабораторными работами по дисциплине, связанной с Java и кроссплатформенными технологиями программирования. В проекте собраны 8 лабораторных работ: от базового синтаксиса Java до коллекций, обработки исключений, регулярных выражений, многопоточности, аннотаций и Reflection API.

## Содержание

| Лабораторная | Тема | Краткое описание |
| --- | --- | --- |
| [Lab1](./Lab1) | Базовые программы на Java | Hello World, поиск простых чисел и проверка палиндромов. |
| [Lab2](./Lab2) | Объектно-ориентированное программирование | Иерархия классов Person, Student, Teacher и TeachingAssistant. |
| [Lab3](./Lab3) | Хэш-таблицы и HashMap | Собственная реализация хэш-таблицы и пример хранения данных о студентах. |
| [Lab4](./Lab4) | Обработка исключений и работа с файлами | Обработка ошибок массива, копирование файлов, пользовательское исключение и логирование. |
| [Lab5](./Lab5) | Регулярные выражения | Поиск чисел, проверка IP и пароля, поиск слов и выделение заглавных букв. |
| [Lab6](./Lab6) | Коллекции и обобщения | Трекер продаж, generic-стек и подсчёт частотности слов в файле. |
| [Lab7](./Lab7) | Многопоточность | Суммирование массива, поиск максимума в матрице и модель склада с грузчиками. |
| [Lab8](./Lab8) | Аннотации, Reflection API и многопоточная обработка данных | Конвейер обработки данных через аннотацию @DataProcessor, reflection и ExecutorService. |

## Структура репозитория

```text
.
├── Lab1/ ... Lab8/        # отдельные лабораторные работы
├── compile_all.sh         # компиляция всех лабораторных работ
├── .gitignore             # исключения для Java/IDE/macOS
├── .editorconfig          # единый стиль форматирования
└── README.md              # описание репозитория
```

В каждой папке лабораторной работы есть собственный `README.md` с кратким описанием, списком файлов, командой компиляции и примерами запуска.

## Требования

- JDK 8 или выше;
- терминал / командная строка;
- любой редактор кода или IntelliJ IDEA.

Проверить установленную версию Java можно командой:

```bash
java -version
javac -version
```

## Быстрый запуск

Скомпилировать все лабораторные работы:

```bash
./compile_all.sh
```

Скомпилировать отдельную лабораторную работу:

```bash
cd Lab1
javac -encoding UTF-8 -d out src/*.java
java -cp out JavaHelloWorldProgram
```

## Примечания

- В репозиторий добавлены только исходные файлы и необходимые текстовые ресурсы.
- Служебные файлы IDE, `.DS_Store`, папки `out` и скомпилированные `.class`-файлы исключены.
- Все лабораторные работы не используют внешние зависимости и запускаются стандартными средствами JDK.
