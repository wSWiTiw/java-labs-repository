# Lab1. Базовые программы на Java

Hello World, поиск простых чисел и проверка палиндромов.

## Состав работы

- `src/JavaHelloWorldProgram.java`
- `src/Palindrome.java`
- `src/Primes.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out JavaHelloWorldProgram
java -cp out Primes
java -cp out Palindrome level madam test
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
