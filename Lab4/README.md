# Lab4. Обработка исключений и работа с файлами

Обработка ошибок массива, копирование файлов, пользовательское исключение и логирование.

## Состав работы

- `src/ArrayAverage.java`
- `src/DivisionLogger.java`
- `src/FileCopier.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out ArrayAverage
java -cp out FileCopier
java -cp out DivisionLogger
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
