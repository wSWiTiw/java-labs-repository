# Lab7. Многопоточность

Суммирование массива, поиск максимума в матрице и модель склада с грузчиками.

## Состав работы

- `src/Task1Sum.java`
- `src/Task2Matrix.java`
- `src/Task3Warehouse.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out Task1Sum
java -cp out Task2Matrix
java -cp out Task3Warehouse
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
