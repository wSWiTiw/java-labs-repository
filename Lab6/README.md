# Lab6. Коллекции и обобщения

Трекер продаж, generic-стек и подсчёт частотности слов в файле.

## Состав работы

- `src/SalesTracker.java`
- `src/Stack.java`
- `src/TopWords.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out SalesTracker
java -cp out Stack
java -cp out TopWords
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
