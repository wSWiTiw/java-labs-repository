# Lab2. Объектно-ориентированное программирование

Иерархия классов Person, Student, Teacher и TeachingAssistant.

## Состав работы

- `src/Main.java`
- `src/Person.java`
- `src/Student.java`
- `src/Teacher.java`
- `src/TeachingAssistant.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out Main
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
