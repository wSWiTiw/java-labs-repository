# Lab3. Хэш-таблицы и HashMap

Собственная реализация хэш-таблицы и пример хранения данных о студентах.

## Состав работы

- `src/HashTable.java`
- `src/Main.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out Main
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
