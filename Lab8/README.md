# Lab8. Аннотации, Reflection API и многопоточная обработка данных

Конвейер обработки данных через аннотацию @DataProcessor, reflection и ExecutorService.

## Состав работы

- `src/Main.java`

## Компиляция

```bash
javac -encoding UTF-8 -d out src/*.java
```

## Запуск

```bash
java -cp out Main
```

> Некоторые программы ожидают ввод с клавиатуры или используют файлы в корне папки лабораторной работы.
